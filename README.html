Fever Report Script

Guia para obtener el archivo de fever client automaticamente.

Consejo: Para ejecutar el script mas rapidamente es mejor copiar y pegar en la maquina local la carpeta de Version 2.1 o extraer el archivo AutoFever.rar localmente en lugar de ejecutarlo directamente desde el disco compartido F:

Archivos Necesarios

El programa funciona extrayendo la informacion de los documentos que obtenemos mediante Jasper.

El programa necesitara los reportes de service availability de las ultimas 4 semanas en formato .csv para generar el reporte.

Sin embargo en esta nueva version el mismo script se encargara de descargar los archivos desde Jasper automaticamente, lo que elimina la necesidad de descargar manualmente los archivos antes de ejecutar el programa.

Adicionalmente se necesita tener el archivo de PERMS.xlsx el la misma carpeta donde se ejecute el programa para que este funcione correctamente

Finalmente se necesita tener instalado Microsoft Edge en el equipo ya que se utilizara para descargar los archivos automaticamente.

Ejecutar Script

Por defecto el archivo se encuentra en la siguiente ruta:

F:\NetEngineering\Group\NSG Team\Analisis Client Issues Services\Autogenerate\Version 2.1

Dentro de esta ruta se encuentra el archivo AutoFever.exe que es el que se encargara de generar el reporte.

Desde esta ruta se puede ejecutar el archivo sin inconvenientes sin embargo al ejecutarse en un disco compartido va a funcionar (mucho) mas lentamente, por lo que es mejor ejecutarlo localmente si es posible.


Ejecutar Script Localmente

Para ejecutar el script localmente se puede copiar y pegar el archivo comprimido AutoFever.rar en el lugar donde se desee ejecutar el programa (en el escritorio por ejemplo) y despues descomprimirlo en esta ubicacion.

El archivo comprimido se puede encontrar en la siguiente ruta

F:\NetEngineering\Group\NSG Team\Analisis Client Issues Services\Autogenerate

Nota: Es posible que el antivirus detecte este archivo como un virus y no permita ejecutarlo para evitar esto habra que agregar el archivo como excepcion dentro del antivirus o alternativamente se puede ejecutar directamente dentro del disco F:.


Funcionamiento

  1. Una vez se ejecute, el programa abrira una linea de comandos y posteriormente se abrira una ventana de Microsoft Edge, se vera que se ejecutan acciones en esta ventana automaticamente, no es necesario hacer nada en esta ventana se puede seguir utilizando el computador o esperar a que finalice de descargar los archivos.

    Nota: Si se ejecuta desde el disco compartido F: es posible que el script se demore unos segundos en abrir la ventana donde pide los archivos, sin embargo, si la linea de comando esta abierta significa que el programa se esta ejecutando. En este caso solo hay que esperar unos segundos a que aparezca la ventana.


  2. Una vez se hayan descargado los 4 archivos el programa creara un archivo previo de excel con la informacion que recopilo de estos archivos de esta forma podemos revisar la informacion antes de generar el archivo final. Si encuentra algun espacio en blanco de donde no pueda obtener informacion se mostrara en la tabla como ---> NO INFO <---habra que revisar el porque este campo no tiene informacion y en caso de hacerse cambios en Jira ejecutar el programa nuevamente para que descargue la informacion actualizada.

    • Se han agregado filtros tambien a las columnas para que se pueda filtrar por campos sin informacion, permanentes en especifico o cualquier valor por el que se desee filtrar

    • En caso de que haya algo que se desee modificar se puede cerrar la linea de comando y modificar la informacion en Jira para generar un nuevo archivo .csv correspondiente a la semana en la que hicimos la modificacion y despues volver a ejecutar la aplicacion.

    • En caso contrario podemos oprimir Enter en el teclado para que el programa genere el archivo final.

    Nota: Si se ejecuta desde el disco compartido F: es posible que el script no registre inmediatamente cuando se oprima la tecla Enter. En este caso probar presionando Enter nuevamente.


  3. En este punto se generara un archivo de excel con el nombre Report from {date}.xlsx. donde el espacio marcado como {date} colocara las fechas de inicio y final del reporte automaticamente. En la linea de comando se indicara la ruta donde fue creado el archivo (siempre es la ruta en donde este el archivo script.exe).

    Para continuar oprimir la tecla Enter.


  4. El programa abrira el archivo para que pueda ser visualizado y modificado, una vez se haya terminado de modificar se puede guardar y cerrar el archivo.

    Nota: Si se ejecuta desde el disco compartido F: es posible que Excel se ejecute pero no abra el archivo. Si esto ocurre habra que abrir el archivo manualmente.


  5. Oprimiendo nuevamente enter en la linea de comando el programa enviara automaticamente el archivo hacia el correo de Network Support para una ultima verificacion, desde aqui se podra reenviar facilmente a los destinatarios que se deseen.


Archivo Final

En esta nueva version el archivo final contara ya con el filtro organizado de mayor a menor en la columna de Total y tambien con las causas raices listadas, solo sera necesario modificar informacion en caso de que se desee y una vez se guarde se enviara automaticamente al correo de Network Support.
